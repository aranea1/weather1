# Weather App

USAGE
---------------------------------------------------------
Deployed at: https://infinite-journey-28108.herokuapp.com/  
Credentials for admin features (username, password): admin password


BEFORE RUN
---------------------------------------------------------
Add mongoDB URI and jwtSecret to config/default.json file



RUN
---------------------------------------------------------
On command line at main folder type in these commands:

        npm start                   runs server
        cd client  npm start        runs client
        
        

RUNTIME
---------------------------------------------------------
node version v12.18.2
npm version 6.14.5
